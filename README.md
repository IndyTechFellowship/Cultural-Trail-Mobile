# Cultural Trail Mobile

### Installing

- Clone repo
  - `git clone git@github.com:IndyXTechFellowship/Cultural-Trail-Mobile.git`
- Use node
  - `npm install`
- Install the React Native CLI
  - `sudo npm install -g react-native-cli`
- Run for Android/iOS
  - `react-native run-android`
  - `react-native run-ios`
